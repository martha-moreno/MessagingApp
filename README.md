# **Messaging App**
*CSE 598 - Software Integration and Engineering*

## Introduction
> This Project is a a simplified messaging service that buffers messages in an XML (or JSON) file and a Web client to
allow users to send and receive messages. 

## Built With
>- .NET
>- C#
